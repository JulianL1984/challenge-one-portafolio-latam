Encriptador

Este proyecto consiste en el desarrollo de un encriptador de palabras utilizando JavaScript, con el objetivo de reforzar los conocimientos en manipulación de cadenas y algoritmos de cifrado básico. La aplicación toma una palabra o frase ingresada por el usuario y la transforma utilizando un método de encriptación simple, en el que cada letra se reemplaza por otra según una regla específica. El encriptador está diseñado para ser intuitivo y fácil de usar. Los usuarios pueden escribir cualquier palabra o conjunto de palabras, y el algoritmo encriptará cada letra desplazándola un número de posiciones en el alfabeto. Se puede implementar una clave personalizada, lo que permite al usuario definir cuántas posiciones se desplazarán los caracteres. Además, el programa incluye una función de desencriptado, que revierte el proceso y permite recuperar el texto original. El proyecto se realizó en JavaScript puro, sin el uso de librerías externas, lo que ofrece una excelente oportunidad para profundizar en la manipulación de arreglos y cadenas. El código es sencillo de integrar en páginas web o aplicaciones más complejas, y está pensado para servir como una herramienta educativa y práctica para desarrolladores en formación.
